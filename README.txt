This datapack was created to analyze the height distribution of ores.

How to use:
All blocks which should be checked must be in the inventory.

Supported blocks are:
	copper ore
	diamond ore
	gold ore
	iron ore
	lapis ore
	lava
	redstone ore

To start the check hold a end rod.

To show the numbers of the found blocks hold
	a copper ingot for copper ore,
	a diamond for diamond ore,
	a gold ingot for gold ore,
	a iron ingot for iron ore,
	a lapis lazuli for lapis ore,
	a lava bucket for lava,
	a redstone for redstone ore,
	a stone for the number of checked blocks for every ore.
